# Docs

Tài liệu được chia theo mục đích nghiên cứu:

- `architecture/`: kiến trúc 1D-CNN/TCAM, tham số, MACs/FLOPs và định hướng tối ưu FPGA.
- `reproduction/`: kế hoạch tái lập, kết quả diagnostic, giả thuyết split/data và lộ trình kiểm chứng.
- `hardware/`: cấu hình server/workstation phục vụ train và benchmark.
